# Fiber Gorm

Simple CRUD Rest Api, created using [GoFiber](https://gofiber.io/ "An Express-inspired web framework written in Go") and [GORM](https://gorm.io/ "The fantastic ORM library for Golang")

## Running the Application

```bash
go run main.go
```
